About the Project

Share2Teach is a file-sharing web app designed for teachers to easily post study material for students. Our aim is to create a platform that facilitates the seamless sharing of educational resources to improve student learning.

Our Project Website

The Share2Teach platform is hosted on an Azure Server, ensuring secure and scalable performance. We developed the backend using Java Spring Boot, IntelliJ, and Maven. For the frontend, we utilized Visual Studio Code with Angular, providing a responsive and dynamic user interface.

About the Founders

We are a group of young, motivated developers with a shared passion for technology and education. This project is the beginning of our journey to make a difference in the IT world by creating innovative solutions. Our goal is to contribute to the educational sector with meaningful projects.
